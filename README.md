# Moroccan Court Kings

The 1v1 street basketball movement born in Marrakech. This repo is the rebranded marketing landing page.

Built with Next.js 15 (App Router) + Tailwind CSS v4. Single-page, deploys to Netlify.

## Quick start

```bash
npm install
npm run dev       # http://localhost:3000
npm run build     # production build
npm run start     # serve production build
```

## Editing copy

All text content (founder names/bios, sponsors, tournament feature blurbs, contact details, next-tournament info) lives in `app/lib/site.ts`. Change it there and every section picks it up.

## Design tokens

Tailwind v4 tokens are defined in `app/globals.css` inside the `@theme` block. Use them via Tailwind utility classes (e.g. `bg-atlas`, `text-cream`, `border-line`) or the CSS variables directly (e.g. `var(--color-atlas)`).

| Token | Color | Use |
|---|---|---|
| `--color-atlas` | `#D7372B` | Primary CTA, accent, headline highlights |
| `--color-atlas-dark` | `#A82319` | Story section background |
| `--color-navy` | `#0E1A2B` | Hero, register, footer |
| `--color-cream` | `#F4EDE0` | Page background, dark-section text |
| `--color-cream-100` | `#FAF5EC` | Tournament section bg |
| `--color-cream-200` | `#EDE4D2` | Sponsors section bg |
| `--color-ochre` | `#E0B354` | Date emphasis, register eyebrow, sponsorship accents |
| `--color-ink` | `#0E1A2B` | Primary text on cream |
| `--color-muted` | `#5C6470` | Secondary text |

Fonts: `Bebas Neue` (display, all H1/H2/H3) + `Inter` (body), loaded via `next/font/google`.

## Adding real assets

Photos and logos are currently styled placeholders. To swap them in:

1. Drop tournament photos into `public/photos/` (4 hero images for the About strip).
2. Drop founder portraits into `public/founders/mr.jpg` and `public/founders/ao.jpg`.
3. Drop sponsor logos as SVGs into `public/sponsors/{slug}.svg`.
4. Replace the styled placeholder divs in `About.tsx`, `Founders.tsx`, and `Sponsors.tsx` with `next/image` tags pointing at the new files.

## Registration form

`Register.tsx` ships with a `mailto:` form so the page is usable on day one. Before launch, swap it for one of:

- **Tally embed** (zero-backend, simplest)
- **Netlify Forms** (add `data-netlify="true"` and a hidden `form-name` input)
- **Formspree / Tally / Typeform** (just iframe the embed)

The form's field shape (name, email, city) is already in place.

## Deploying to Netlify

1. Push this repo to GitHub.
2. In Netlify, "Add new site → Import from GitHub" and pick the repo.
3. Build settings come from [netlify.toml](netlify.toml) — no manual config needed.
4. Once live, point `moroccancourtkings.com` (and/or `.ma`) at the Netlify site under "Domain management".

## SEO / sharing

- Page title, description, OG, and Twitter card are set in [app/layout.tsx](app/layout.tsx).
- A designed 1200x630 OG image is generated by [app/opengraph-image.tsx](app/opengraph-image.tsx) at `/opengraph-image`.
- `SportsOrganization` JSON-LD is injected from the root layout.
- `robots.txt` and `sitemap.xml` are auto-generated.

## Accessibility

- "Skip to content" link at the top of every page.
- Sticky nav supports keyboard focus; mobile menu announces expanded/collapsed.
- Color contrast meets WCAG AA on the cream/navy/atlas combinations used for text.
- `prefers-reduced-motion` is honored (animations are disabled).
- Every interactive element has an `aria-label` or visible text.

## Brand rules

Read [BRAND.md](BRAND.md) before changing copy, names, or colors. The biggest don'ts:

- Don't bring back the Gothic blackletter wordmark or the umlaut gimmick.
- Don't use emoji as UI icons. Use `lucide-react`.
- Don't write SaaS-template copy. ("Premier basketball tournament management system." was the old footer. It's gone for a reason.)
- The founder's name is **Abdellah Oueldeghzal** — not "Abdollah Oueldleghzal". One spelling, everywhere.
